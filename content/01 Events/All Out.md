# All Out

**Type:** Rotating "Kill Event"
**Priority:** 🟠 High — major rewards, best paired with Training Events

## What it is
A Kill Event — points are earned by defeating enemy troops (through gathering monster hunts, rallies, garrisons, or PvP depending on the specific window's rules). Also referenced as occurring shortly before [[Champagne Fair]] in the monthly cycle.

## Core Strategy
- Maximize kill count during the active window rather than spreading activity evenly across the whole event period
- Time any planned troop training around this window if it overlaps a Training Event — training completions during Kill/Training windows earn bonus event points
- Coordinate with alliance on which targets to hit (bosses/monsters vs PvP) based on that specific run's rules

## Prep checklist
- [ ] Check the specific kill-source rules for this run (monsters vs PvP vs both)
- [ ] Hold speedups if a Training Event overlaps
- [ ] Heal up beforehand so you're not wasting window time on recovery

## Teaching point for new members
> All Out is a Kill Event — the more enemies/monsters you defeat during the window, the more points you get. Check what counts this time (it varies), and don't waste the window healing up beforehand — do that in advance.
