# Armament Competition

**Type:** Rotating 2-day dual-type individual event
**Priority:** Medium-High — timing your material spending around this window matters

## What it is
A 2-day event focused on gear and Truegold materials, split into two scoring "types" active on different days.

## The Two Types
- **Type 1** — scores Hero Shard usage. Save Hero Shards for this window instead of using them piecemeal.
- **Type 2** — scores Truegold and Hero Gear material usage. Save Truegold and Hero Gear materials for this window.

Check which type is active before spending — using the wrong material on the wrong day earns far less event value than timing it correctly.

## Prep checklist
- [ ] Stockpile Hero Shards between events for Type 1 days
- [ ] Stockpile Truegold/Hero Gear materials for Type 2 days
- [ ] Check the event's Score Builder/Event Guide tab in-game to confirm which day is which type

## Common mistakes
- Spending Hero Shards or Truegold outside a matching Armament Competition window
- Not checking which type (1 or 2) is live before dumping materials

## Teaching point for new members
> Armament Competition has two types on two different days — one rewards Hero Shards, the other rewards Truegold/Hero Gear materials. Check which one is live before you spend anything, and save materials between events specifically for this window.
