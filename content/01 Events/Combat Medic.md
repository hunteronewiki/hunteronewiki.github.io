# Combat Medic

**Type:** Rotating individual event (part of the 4-week cycle)
**Priority:** Medium

## What it is
An event centered on healing/recovery activity — treating wounded troops and related actions score points during this window.

## Strategy
- Time troop healing around this window if you have wounded troops piling up from other events (KvK, Viking Vengeance, etc.)
- Don't rush to heal outside this window if you can hold off — healing during Combat Medic earns bonus event points on top of the normal recovery

## Teaching point for new members
> Combat Medic rewards healing activity — if you've got wounded troops sitting around, hold off and heal them during this window instead of earlier, so it counts toward the event.
