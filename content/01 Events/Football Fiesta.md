# Football Fiesta

**Type:** Seasonal minigame event
**Priority:** Medium — fun side event with its own reward track

## What it is
A seasonal event built around a shooting minigame ("Green Arena"), with its own battle pass, ranking system, and reward shops.

## Key Features
- **Green Arena** — the core shooting minigame that earns you Energy Bar-based currency
- **Energy Bar farming** — the main resource to manage; most players miss a consistent farming routine for it, so check daily rather than playing in one big burst
- **GOAT Prize** — a top-tier milestone reward
- **Auracle Wish Lodge** — a separate reward shop/wish system tied to the event
- **Coach of Legend ranking** — the competitive leaderboard track
- **Football battle pass** — a separate premium/free track running alongside the main event

## Prep checklist
- [ ] Farm Energy Bar consistently every day rather than saving it all for one session
- [ ] Check the shop priority for your spender type (F2P vs paying) before buying anything
- [ ] Track battle pass progress separately from the main event track

## Teaching point for new members
> Football Fiesta is a fun seasonal minigame — farm your Energy Bar daily instead of letting it sit unused, and check the shop before spending on anything specific.
