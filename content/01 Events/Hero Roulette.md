# Hero Roulette

**Type:** Solo gacha-style hero summoning event
**Priority:** 🟡 Medium-High — main way to target specific hero shards with gems

## What it is
A targeted summoning system letting you spend gems on spins aimed at specific heroes, rather than the standard random Advanced/Epic Recruitment pulls.

## Core Strategy
- **Use 10-pulls, not single pulls** — 10-pull batches are the efficient way to spend here; single spins waste value
- **Time it with [[VIP System|VIP]] activation** if you're planning to activate/renew VIP status — stacking bonuses together maximizes value
- **Save keys/gems for this specifically** rather than the general recruitment pool if you're chasing one particular hero
- Best paired with a top-up/spend window if you're a spender, since this is the most gem-efficient way to target specific hero shards

## Prep checklist
- [ ] Decide which hero you're targeting before spinning (don't pull randomly)
- [ ] Batch your pulls in 10s
- [ ] Time any gem top-ups around this event for maximum efficiency

## Common mistakes
- Spinning one at a time instead of batching into 10-pulls
- Pulling without a specific hero target in mind

## Teaching point for new members
> Hero Roulette is how you target a specific hero's shards with gems — always pull in batches of 10, and know which hero you're going for before you start spinning instead of just pulling randomly.
