# Officer Project

**Type:** Rotating dual-type individual event
**Priority:** Medium-High — pairs with [[Armament Competition]] as a materials-timing event

## What it is
A dual-type event similar in structure to Armament Competition, split into a Forgehammers-focused type and a Charm Designs-focused type.

## The Two Types
- **Charms/Troops type** — scores Charm material and troop-training activity
- **Hero Shards/Hero Gear type** — scores Hero Shard and Hero Gear material usage

As with Armament Competition, check which type is active before spending — save the relevant materials between events so you're ready.

## Prep checklist
- [ ] Save Charm materials for the Charms/Troops day
- [ ] Save Hero Shards and Hero Gear materials for the other day
- [ ] Confirm the active type in-game before spending

## Teaching point for new members
> Officer Project works like Armament Competition — two types, two different material focuses. Check which type is live, then spend the matching materials you've been saving up.
