# Strongest Governor

**Type:** Major rotating solo/individual competitive event
**Priority:** 🔴 Highest — one of the two biggest reward events alongside [[Kingdom vs Kingdom (KvK)]]

## What it is
A multi-week-cycle event that rewards individual Governor progress — training, construction, [[Research and Academy|research]], and combat activity all typically count toward this event's point categories depending on that cycle's theme.

## Key strategy: Save speedups for the matching day
Strongest Governor cycles through different focus categories (commonly training, construction, research). **Save your speedups and resources for the specific day/category that matches what you're strong in** — using them outside the matching window earns dramatically less event value.

## Prep checklist
- [ ] Check which category is active before dumping speedups
- [ ] Queue up building/research/training items in advance so you can complete them during the matching window rather than starting cold
- [ ] Don't blow speedups on off-category days "just because you have them"

## Common mistakes
- Using speedups the moment you get them instead of holding for the matching Strongest Governor day
- Not queuing anything in advance, so the event window opens with nothing ready to complete

## Teaching point for new members
> Strongest Governor rewards individual progress — but only if you time it right. Hold onto speedups until the day that matches training/building/research, and have things already queued up so you can finish them the moment the matching window opens.
